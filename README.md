# AI Job Recommendation System

An AI-powered job recommendation system that suggests suitable job roles based on user skills using Machine Learning techniques.

---

## Project Overview

The AI Job Recommendation System helps users identify suitable job opportunities by analyzing their skills and matching them with relevant job descriptions.

The application provides personalized recommendations through a simple web interface built with Streamlit.

---

## Features

- Skill-based job recommendations
- AI-powered recommendation logic
- Interactive Streamlit interface
- CSV dataset integration
- Simple and user-friendly design

---

## Tech Stack

- Python
- Streamlit
- Pandas
- Machine Learning

---

## How to Run

1. Clone the repository

```
git clone https://github.com/pallavi-karna/job-recommender.git
```

2. Install dependencies

```
pip install pandas streamlit
```

3. Run the application

```
streamlit run app.py
```
